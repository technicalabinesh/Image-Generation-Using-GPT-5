# Image Generation Using GPT-5

This repository contains a Jupyter Notebook that generates an image using Azure OpenAI (`gpt-image-2`) and saves it as `generated_image.png`.

## File

- `Image_Generation_Using_GPT_5.ipynb` — notebook with the complete image-generation flow

## Requirements

- Python environment with Jupyter support
- Azure OpenAI endpoint
- Azure OpenAI API key

The notebook installs:

- `openai`
- `pillow`

## How to run

1. Open `Image_Generation_Using_GPT_5.ipynb`.
2. Run the notebook cells.
3. Enter your Azure OpenAI API key when prompted.
4. Update endpoint/model values in the notebook if needed for your deployment.
5. Find the generated output image as `generated_image.png`.
